# LibraryDB-Advanced-Database-System
E-Library Management System
-LibraryDB SQL Server database for a university library system. 
-Built for CT004-3-3 Advanced Database Systems. 

## Features 
-normalized tables (up to 3NF) 
-Stored Procedures 
-Triggers
-Constraints
-Indexing, Views
-Table Partitioning 
-analytical SQL queries
